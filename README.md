# 英語学習アプリ
自身のペースで英語学習ができるアプリケーションです。\
英語を学びたいと思っている人は誰でも取り組むことができます。\
以下に画面ごとの説明を記載しています。
## ログイン画面
メールアドレスとパスワードを入力してログインできる。
## 新規登録画面
ユーザー名、メールアドレス、パスワードを入力してアカウント登録をする。
## ホーム画面
![ホーム画面](C:\Users\user\Pictures\Screenshots/ホーム画面.png "サンプル")
・カレンダー(左右の矢印で月の切り替えが可能)\
・日付表示\
・学習問題数表示(カレンダーの日付にカーソルを合わせる)\
・学習問題数による日付の色分け\
・問題演習ボタン(通常モードと復習モード)\
・各種設定画面への遷移ボタン\
・ログアウトボタン
## 通常学習モード
・学習問題数の選択(5～20まで選択可能)\
・出題範囲の指定(すべての問題or未回答のみ)
## 復習モード
・復習問題が何問あるかの表示\
・取り組む問題数の選択\
・復習問題の総数が0問の時はスタートボタンが押せない。
## 問題演習
・選択した問題数だけ英文と4択問題が出題される。\
・それぞれの英文に対して最も適切な選択肢を選ぶ。\
・すべての問題にチェックし終わったら、最下部の回答ボタンを押す。
## 回答と解説
・各問題に対して正解と不正解および解説を表示する。\
・正解は青、不正解は赤の文字で表示される。\
・確認が終わったら最下部の「ホームへ戻る」ボタンを押すと解いた問題としてカウントされる。
## 各種設定画面
・パスワード変更\
・メールアドレス変更\
・アカウント削除\
・ログアウト
